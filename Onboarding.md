# Onboarding

Onboarding is a multistep process for onboarding the app and the ovale ecosystem.

## Process

* User enters email
* if email is not found
    * ask for a name
    * we ask for a two step password
    * we send a validation email
    * validation is true
    * jmp authenticate
* if email found
    * stp authenticate
